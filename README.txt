The method accepts a string, which is a mathematical expression.		
Returns the result of this expression.								
The method works with such mathematical operators and functions:		

! (factorial)																
sin cos tg ctg arcsin arccos arctg arcctg								
^   sqr log lg ln														
*   /   //  %															
+   -																	
Perceives the number "pi" and "e"
